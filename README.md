# goseland.org

Portfolio site for Jeff Goseland. Built with Astro, deployed to Netlify.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Push to `main` on GitHub. Netlify auto-deploys to [goseland.org](https://goseland.org).

Sub-path proxies for other projects are configured in `netlify.toml`.
